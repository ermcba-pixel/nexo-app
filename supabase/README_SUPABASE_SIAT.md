# nexo - Supabase Facturación Masiva Bolivia / SIAT

Primero ejecutar en Supabase:
supabase/01_supabase_facturacion_masiva_bolivia_siat.sql

Este script prepara:
- configuración SIAT del sistema nexo 1.0.1
- facturación electrónica en línea
- emisión masiva
- CUIS / CUFD / CUF
- catálogos SIAT
- facturas, items, lotes, contingencias
- registro ventas IVA
- cierres fiscales
- auditoría fiscal
- regla NIT 99001 para clientes fuera de Bolivia
